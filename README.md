# SmileyDrip
the best quality merch

## Local preview

Open `index.html` in a browser or run a simple static server from the project root:

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

The site files are:
- `index.html` — main page
- `css/styles.css` — styling
- `assets/black-hoodie.svg`, `assets/white-hoodie.svg` — placeholder images

Replace the PayPal links in `index.html` with your account link before publishing.
